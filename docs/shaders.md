# Shaders implementation

## Overview

Create the following shader types as layers:

- [color correction](https://maximmcnair.com/p/webgl-color-correction)
- [hue](https://maximmcnair.com/p/webgl-hue)
- [blend modes](https://maximmcnair.com/p/webgl-blend-modes)
- [film grain](https://maximmcnair.com/p/webgl-film-grain)
- [duotone](https://maximmcnair.com/p/webgl-duotone)
- [pixelate](https://maximmcnair.com/p/webgl-pixelate)
- [thresholding](https://maximmcnair.com/p/webgl-thresholding)
- [dithering](https://maximmcnair.com/p/webgl-dithering)
- [vignetting](https://maximmcnair.com/p/webgl-vignette)
- [chromatic aberration](https://maximmcnair.com/p/webgl-chromatic-aberration)
